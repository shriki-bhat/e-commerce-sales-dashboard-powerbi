# E-Commerce Sales Performance Dashboard (Power BI)

This project is a simple yet effective E-Commerce Sales Performance Dashboard built in **Power BI**, using a sample dataset provided by Aditi Analytics (TechTip24).

## Objective
Analyze revenue, profit, and regional performance to practice data analytics and financial insight generation as a **MERN Stack Developer, Data Analyst, and Finance Enthusiast**.

## Files
- `E-Commerce Sales Dashboard.pbix` – Main Power BI report.
- `E-Commerce Data.xlsx` – Source dataset.
- `dashboard_preview.png` – Screenshot of the dashboard.

## Key Insights
- Total Revenue: ~$12.64M across multiple product categories and regions.
- APAC and EU are top markets by revenue.
- United States contributes the highest share among countries.
- Standard Class is the most used delivery mode.

## Tools & Skills
- Power BI Desktop
- Power Query
- Excel Data Modeling
- Data Visualization & Financial Analysis
